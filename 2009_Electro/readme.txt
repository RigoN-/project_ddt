Системные требования:
CPU - P4/AthlonXP, ОЗУ - 256 MB, Video 64 MB совместимая с OpenGL 1.5
Windows:
Для запуска программы скопировать каталог с программой на жесткий дис, запускаем на выполнение -  SDL_phisic_demo.exe.
Linux:
Для запуска программы скопировать каталог с программой на жесткий дис, запускаем на выполнение -  SDL_phisic_demo
Основные клавиши управления программой:
F1-F12-переключение опытов;
A-разработчики;
1-5-переключение схем;
I-отобразить/скрыть информацию;
H-справка;
O-вкл/выкл;
+/-(дополнительной блок цифровой клавиатуры ) - движение ползунка реостата;
Z-отобразить/скрыть описание опыта;
Esc-выход.
