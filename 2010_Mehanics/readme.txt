SDL_Phi_0_003 - программа реализующая трехмерный симулятор лабораторных работ по разделу физики - Механика.

Системные требования:
ЦПУ - PIV, Athlon XP или новее;
Видеоадаптер - 128 Mb, Shader Model 3.0, OpenGL 2.1;
ОЗУ - от 512;
ОС - Linux и установленные драйвера с поддержкой  3D, разрешение экрана не менее 1024х768.
для работы требуются установленные библиотеки SDL, SDL_ttf, SDL_image.

ОС - Windows и установленные драйвера с поддержкой  3D, разрешение экрана не менее 1024х768.

Для запуска в Linux:
зайти в каталог bin и запустить на исполнение:
./SDL_Phi_0_003

Для запуска в Windows:
зайти в каталог bin и запустить на исполнение:
SDL_Phi_0_003.exe

Для сборки в Linux:
зайти в каталог src и выполнить:
make -f Makefile.sdl

Для сборки в Windows:
зайти в каталог src и выполнить:
mingw32-make -f Makefile.sdl.win

Управление в программе:
Переключение лабораторных работ F1-F6;
Переключение опытов 0-9;
Оразработчиках - F11
Руководство по работам - F12
Управление камерой - зажатть ЛКМ и перемещать указатель
Приблизить/отдалить - Колесо мыши
Переключение шейверов - S
Выход - Esc;
Поворот плоскости - зажать левую кнопку мыши и перемещать указатель.
