math_function - программа демонстрирующая графики функций.

Системные требования:
ЦПУ - PIV, Athlon XP или новее;
Видеоадаптер - 128 Mb, Shader Model 3.0, OpenGL 2.0;
ОЗУ - от 512;
ОС - Linux и установленные драйвера с поддержкой  3D, разрешение экрана не менее 1024х768.
для работы требуются установленные библиотеки glut.
Структура проекта:
bin - испольняемые файлы, необходимые библиотеки.
doc - описание проекта.
src - исходные тексты проекта.
data - ресурсы проекта: текстуры,иконки.


Для запуска в Linux:
зайти в каталог bin и запустить на исполнение:
./gmathematics

Для запуска в Windows:
зайти в каталог bin и запустить на исполнение:
gmathematics.exe

Для сборки в Linux:
зайти в каталог src и выполнить:
make -f Makefile.linux

Для сборки в Windows:
зайти в каталог src и выполнить:
make -f Makefile.win

Управление в программе:
Для смены функций F1-F12;
Выход - Esc;
Поворот плоскости - клавиши стрелок.
