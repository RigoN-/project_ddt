Landscape - программа, демонстрирующая возможность создания ландшафта при помощи карты высот.

Для запуска данной программы потребуется наличие в системе следующих библиотек:
GLUT;
OpenGL.

Файлы, необходимые для запуска:
landscape - исполняемый файл;
texture.bmp - файл-изображение текстуры;
heightmap.bmp - файл-изображение карты высот.

Управление:
Стрелка вверх - вращение по оси x (вниз);
Стрелка вниз - вращение по оси x (вверх);
Стрелка влево - вращение по оси y (влево);
Стрелка вправо - вращение по оси y (вправо);

Запуск:
Через консоль. Команда "./landscape" без кавычек.
